arc.parser¶

A module for parsing information from various files.

arc.parser.parse_dipole_moment(path)[source]¶

Parse the dipole moment in Debye from an opt job output file.

arc.parser.parse_e_elect(path, zpe_scale_factor=1.0)[source]¶

Parse the electronic energy from an sp job output file.

Parameters

  • path (str) – The ESS log file to parse from.

  • zpe_scale_factor – The ZPE scaling factor, used only for composite methods in Gaussian via Arkane.

Returns

The electronic energy in kJ/mol

Return type

e_elect (float)

arc.parser.parse_frequencies(path, software)[source]¶

Parse the frequencies from a freq job output file.

arc.parser.parse_polarizability(path)[source]¶

Parse the polarizability from a freq job output file, returns the value in Angstrom^3.

arc.parser.parse_scan_energies(path)[source]¶

Parse the torsion scan energies from an ESS log file.

Parameters

path (str) – The ESS log file to parse from.

Returns

The electronic energy in kJ/mol. angles (list): The scan angles in degrees.

Return type

energies (list)

arc.parser.parse_t1(path)[source]¶

Parse the T1 parameter from a Molpro coupled cluster calculation.

arc.parser.parse_xyz_from_file(path)[source]¶

Parse xyz coordinated from: - .xyz: XYZ file - .gjf: Gaussian input file - .out or .log: ESS output file (Gaussian, QChem, Molpro) - other: Molpro or QChem input file

Parameters

path (str) – The file path.

Returns

The parsed coordinates.

Return type

xyz (dict)

Determine the calculated ZPE from a frequency output file

Parameters

path (str) – The path to a frequency calculation output file.

Returns

The calculated zero point energy in kJ/mol.

float

arc.parser.process_conformers_file(conformers_path)[source]¶

Parse coordinates and energies from an ARC conformers file of either species or TSs.

Parameters

conformers_path (str) – The path to an ARC conformers file (either a “conformers_before_optimization” or a “conformers_after_optimization” file).

Returns

Entries are conformer coordinates in a dict format.

Return type

xyz (list)

Returns

Entries float numbers representing the energies in kJ/mol.

Return type

energies (list)

Raises

InputError – If the file could not be found.
